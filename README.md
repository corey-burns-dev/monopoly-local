# Monopoly Local

Monopoly Local is a browser-based Monopoly-style game for shared-device play. It supports full local matches with turn flow, property management, cards, jail, auctions, and win detection.

## Highlights

- Supports `2` to `8` local players
- End-to-end board game rules and turn handling
- Shared-screen play with React state and animated UI

## Stack

- React
- TypeScript
- Vite

## Quick start

```bash
npm install
npm run dev
```

Tests and build:

```bash
npm test
npm run build
```
